1. Inside /project1 dir --> run command node app (or nodemon app) to run server
2. Search localhost on browser until webpage pops up.
3. Click "add contact" page to add a users. You will be directed to the users page with your user added. 
4. If using nodemon, terminal will show the updated user list as they are added. 
5. In another terminal enter command "mongod" to run database. 
6. Finally in root directory (same dir as app.js) enter command node consume_api.
